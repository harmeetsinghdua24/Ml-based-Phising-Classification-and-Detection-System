## Overview
The **ML-based Phishing Classification and Detection System** is a cybersecurity-focused project developed to identify and classify phishing websites using Machine Learning techniques. The project aims to spread awareness about online scams and demonstrate how phishing attacks can be detected and prevented effectively.

This system analyzes website-related features and predicts whether a website is **legitimate** or **phishing** using a trained Machine Learning model.

---

## Features
- Detection of phishing websites using Machine Learning
- Classification of URLs as safe or malicious
- User-friendly and lightweight implementation
- Awareness about phishing attacks and cyber frauds
- Fast prediction using trained ML models

---

## Technologies Used
- Python
- Machine Learning
- Scikit-learn
- Pandas
- NumPy
- JSON
- VS Code

---

## Project Objectives
- To detect phishing websites accurately
- To improve cybersecurity awareness
- To reduce risks caused by fake websites and login pages
- To demonstrate the use of ML in cybersecurity applications

---

## Working of the System
1. The dataset containing website features is collected.
2. Data preprocessing and cleaning are performed.
3. Machine Learning algorithms are trained on the dataset.
4. The trained model predicts whether a website is phishing or legitimate.
5. Results are displayed to the user.

---

## Machine Learning Approach
The project uses supervised Machine Learning techniques for phishing detection. Different website parameters and URL-based features are analyzed to classify websites.

### Possible Features Used
- URL length
- Presence of special characters
- HTTPS usage
- Domain age
- Suspicious redirects
- IP address usage
